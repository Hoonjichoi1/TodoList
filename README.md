# Todo List
A task management app built with React and Firebase.

🔗 [Live Demo](https://todo-list-seven-gamma-41.vercel.app)

<img width="551" height="680" alt="Screenshot 2026-03-22 at 19 13 02" src="https://github.com/user-attachments/assets/2019e6a5-8fc9-4bf4-8818-e3a40896160a" />


## Features
- Add, edit, and delete tasks
- Set due dates with automatic priority indicators (red:within 3days / yellow: 7days / green)
- Mark tasks as complete with strikethrough
- Search tasks by keyword
- Filter by status — All / In Progress / Completed
- Completed tasks sorted to the bottom
- Tasks sorted by due date automatically
- Data persisted with Firebase Firestore in real time

## Tech Stack
- JavaScript, React, CSS, HTML
- Firebase Firestore
- Vercel

## Getting Started
```bash
npm install
npm run dev
```
